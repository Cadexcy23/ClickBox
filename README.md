# ClickBox
Description: Box with little jumping mochi creatures to click on.\
Author: Caden Metzgus\
Created: 10/13/2018\
Platform: Windows

## Controls
1 2 3 - Spawn creature at the mouse position.\
Left Click - Clear any creatures around the mouse.
